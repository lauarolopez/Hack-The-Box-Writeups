# 📝 Personal Notes

Esta carpeta contiene mis apuntes personales sobre ciberseguridad.

A diferencia de los write-ups de las máquinas, aquí documento conceptos, teoría y explicaciones que quiero recordar a largo plazo.

## 🎯 Objetivo

Construir una base de conocimiento propia que pueda consultar en cualquier momento sin depender de buscar información nuevamente.

## 📂 Organización

Los apuntes estarán organizados por temática.

Ejemplos:

- 🐧 Linux
- 🪟 Windows
- 🌐 Networking
- 📡 Active Directory
- 🌍 Web Security
- 🔍 Enumeration
- 🔑 Privilege Escalation
- 📂 File Transfers
- 🛠️ Tools
- 📜 Scripting

## 📖 Cada nota incluirá

- Explicación del concepto.
- Casos de uso.
- Ejemplos.
- Buenas prácticas.
- Referencias útiles.
