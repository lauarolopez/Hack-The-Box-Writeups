# 🛡️ Hack The Box Writeups

> Documentando mi aprendizaje en ciberseguridad, pentesting y Hack The Box.

---

# 👋 Bienvenido

Este repositorio es mi base de conocimiento personal mientras avanzo en el mundo de la ciberseguridad.

Mi objetivo no es solamente completar laboratorios, sino entender cómo funcionan las vulnerabilidades, cómo identificarlas y cómo mitigarlas.

Toda la documentación está escrita por mí como parte de mi proceso de aprendizaje.

---

# 🎯 Objetivos

- 📚 Aprender ciberseguridad de forma práctica.
- 🔍 Mejorar mis habilidades de enumeración.
- ⚡ Comprender cada vulnerabilidad antes de explotarla.
- 🧠 Documentar todo lo aprendido.
- 📈 Crear una base de conocimiento para futuras consultas.
- 💼 Construir un portfolio técnico.
