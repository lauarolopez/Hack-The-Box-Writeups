# 📚 Hack The Box Academy

Bienvenido a mi sección de **Hack The Box Academy**.

Aquí documento todos los módulos que completo mientras aprendo ciberseguridad.

## 🎯 Objetivo

Mi objetivo no es únicamente terminar los módulos, sino comprender cada concepto y poder aplicarlo en laboratorios reales.

## 📂 Organización

Cada módulo tendrá su propia carpeta con:

- 📖 README.md (documentación completa)
- 📝 notes.md (apuntes personales)
- ⚡ commands.md (comandos utilizados)
- 📸 screenshots/ (capturas de pantalla)
