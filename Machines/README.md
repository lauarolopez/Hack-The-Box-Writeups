# 🖥️ Hack The Box Machines

Bienvenido a mi colección de **Hack The Box Machines**.

Aquí documento todas las máquinas que completo, organizadas por nivel de dificultad.

## 🎯 Objetivo

El objetivo de esta sección es mejorar mis habilidades de pentesting, enumeración, explotación y escalada de privilegios documentando cada paso del proceso.

## 📂 Organización

Las máquinas están organizadas por dificultad:

- 🟢 Easy
- 🟡 Medium
- 🔴 Hard
- ⚫ Insane

Cada máquina tendrá su propia carpeta con la siguiente estructura:

```text
Machine-Name/
│
├── README.md
├── notes.md
├── commands.md
└── screenshots/
```

## 📖 ¿Qué contiene cada máquina?

### 📄 README.md

- Información de la máquina
- Objetivo
- Enumeración
- Explotación
- Privilege Escalation
- Flags obtenidas
- Conceptos aprendidos
- Thinking Process

### 📝 notes.md

Apuntes personales, dudas, ideas e investigaciones relacionadas con la máquina.

### ⚡ commands.md

Todos los comandos utilizados durante la resolución.

### 📸 screenshots/

Capturas ordenadas cronológicamente.

Ejemplo:

```text
01-nmap.png
02-enumeration.png
03-exploit.png
04-privesc.png
05-root.png
```

## 🧠 Filosofía

Mi objetivo no es simplemente resolver máquinas.

Quiero entender:

- Por qué funciona un ataque.
- Qué vulnerabilidad se está explotando.
- Cómo podría detectarse.
- Cómo podría mitigarse.
- Qué aprendí durante el proceso.
